# CFPB-Text-Analysis-Using-NLP-Techniques


In this task I will use NLP techniques in Python to analyse texts comprising complaints regarding debt collection, credit information, mortages, and student loans. The data is unstructured, not allowing for direct systematic analysis. The goal is to extract the most frequently addressed topics from the texts.


This repository contains the following files:
- The csv file of the complaints I used for this project
- The project file (task 1) contains the fully functional code, together with comments which provide a brief technical description on the code
- The data exploration file where I explored the data and thoughts I had from phase 2 before completeing the project
